# 📌 Project Overview

This repository shows how to set up a **basic development environment** required for modern web development.

The environment includes:

✔ Installing **Node.js**  
✔ Creating a Node project using **npm**  
✔ Writing a **JavaScript program**  
✔ Running the program using **Node.js**  
✔ Using **Visual Studio Code Terminal**
